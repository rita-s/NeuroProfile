# 🧠 NeuroProfile

**NeuroProfile** is a self-report cognitive architecture screener designed to map traits across four domains:

- 🟠 Autism  
- 🔴 ADHD  
- ⚫ Giftedness  
- ⚪ Overlap  

It visualizes cognitive distribution using a radar model and structured profile logic.

> ⚠️ This is NOT a clinical diagnosis.  
> It is a reflective profiling tool.

---

## ✨ What It Does

NeuroProfile:

- 📊 Maps 40 structured self-report items (0–10 scale)
- 🧭 Calculates category intensity (0–100%)
- 🕸 Generates radar visualization (SVG-based)
- ⚡ Identifies potential strengths ("Superpowers")
- 💥 Flags friction points ("Kryptonite")
- 🧠 Produces a rule-based profile summary

Built as a **cognitive pattern mapper**, not a medical instrument.

---

## 🏗 Architecture

The system is divided into four domains:

| Domain | Description |
|--------|------------|
| 🟠 Autism | Pattern depth, structure preference, sensory profile |
| 🔴 ADHD | Energy regulation, novelty drive, working memory |
| ⚫ Giftedness | Systems thinking, abstraction, cross-domain synthesis |
| ⚪ Overlap | Shared neurodivergent intensity traits |

### Scoring Model

Output includes:

- Percentage intensity per domain
- Radar visualization
- Threshold-based profile classification
- Trait-level 10/10 highlights

---

## 📈 Radar Model

The radar chart visualizes:

- ⚫ Giftedness (Top axis)
- ⚪ Overlap (Right axis)
- 🔴 ADHD (Bottom axis)
- 🟠 Autism (Left axis)

Provides a structural overview of cognitive distribution.

---

## 🧩 Profile Classification

Profiles are threshold-driven and transparent.

Possible outputs:

- 🧠 Triple-Exceptional (3e)
- 🧠 Twice-Exceptional (2e)
- ⚫ Gifted-Dominant
- 🔴 ADHD-Dominant
- 🟠 Autism-Dominant
- ⚪ High-Overlap
- 🌱 Emerging Profile

Classification logic is deterministic and editable.
